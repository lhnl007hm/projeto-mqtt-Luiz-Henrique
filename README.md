# Entregável Sprint 8

**Objetivo

O objetivo desse projeto é utilizar um Arduino Uno mais um Sensor Magnético para monitorar se a porta de um Rack de Rede está **ABERTO** ou **FECHADO**; enviar essa informação via internet utilizando o protocolo MQTT (*Message Queuing Telemetry Transport*) para um servidor MQTT hospedado na *Amazon Web Service* (AWS) e exibir a informação em um cliente MQTT [Flex Grid] (https://play.google.com/store/apps/details?id=net.routix.mqttdash&hl=pt_BR&gl=US) instalado em um Smartphone, conforme abaixo.
